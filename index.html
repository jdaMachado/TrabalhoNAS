Uso Doméstico/Pessoal e Pequenos Escritórios:

QNAP TS-128A

  • Preço: 139,90 €
  • Processador: Realtek RTD1295 quad-core de 1,4 GHz
  • Memória RAM: 1GB DDR4, não expandivel
  • Portas USB: Duas portas USB 2.0 e uma porta USB 3.0
  • Unidades de armazenamento: Suporte para uma unidade de armazenamento
  • Portas de Rede: Uma porta de rede 1GbE (RJ45)
  • Silencioso e económico – Consumo energético 7,06 W
  • Serviço de Cloud privado
  • Software ‘Hybrid Backup Sync’ permite configurar funcionalidades de Backup

Synology DS418

  • Preço: 439,90 €
  • Processador: Realtek RTD1296 quad-core Realtek 1,4 GHz
  • Memória RAM: 2GB DDR4
  • Portas USB: Duas portas USB 3.0
  • Unidades de armazenamento: Suporte para quatro unidades de armazenamento
  • Portas de Rede: Duas portas de rede 1GbE (RJ45)
  • Bom desempenho em tarefas de leitura e escrita (226 MB/s e 170MB/s sequenciais)
  • Servidor multimédia de 4K: Suporte para transcoding de 10-bit 4K H.265
  • Tecnologia Synology Hybrid Raid (SHR): Facilita a configuração RAID e permite redundância até 2 discos para salvaguardar a informação em caso de danificação dos discos.
